Project Overview
----------------

MS-dTMT is a mobile application developed with Flutter for cognitive testing. The project implements cognitive assessment tools such as the Digital Trail Making Test (Digital TMT),
aimed at providing a convenient and reliable solution for cognitive function assessment.
